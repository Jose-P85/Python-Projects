# Python-Projects
My python projects

Project 1: Tortoise and Hare Probability Simulation

Summary:
This program simulates the classic tortoise and hare race 100,000 times. The simulation uses random movement and sleep patterns to determine which racer wins more often.

Key Features:

Simulates 100,000 races.
Random movement for both racers.
Tracks total wins for each racer.

Project 2: Movie Recommendation System

Summary:
This program recommends movies based on a user's preferred genre and minimum rating. It searches a movie dataset and provides matching recommendations.

Key Features:

Searches movies by genre.
Filters movies by rating.
Gives up to three recommendations.

Project 3: Number Guessing Game

Summary:
This interactive game challenges players to guess a randomly generated number within five attempts. Players can choose between different difficulty levels.

Key Features:

Three difficulty levels.
Five guess limit.
Hints after incorrect guesses.

Project 4: Slot Machine Simulator

Summary:
This program simulates a slot machine game where players can add credits, spin the reels, and win prizes based on symbol combinations.

Key Features:

Credit system.
Random slot spins.
Multiple winning combinations.
Jackpot reward.
